# FindAFriend

A friend finding app that finds your closest match from the cast of Critical Role!

## Getting Started

To run locally download the files, install the node modules. Type node server to launch the server. Otherwise visit https://criticalrolefriends.herokuapp.com/ to take the survey!

### Prerequisites

Node.js and required modules

## Built With

* Javascript - Main logic
* [Node.js](https://nodejs.org/en/) - JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js' package ecosystem, npm, is the largest ecosystem of open source libraries in the world.
* [epress](https://www.npmjs.com/package/express) - Node module used to establish my server.
*[body-parser](https://www.npmjs.com/package/body-parser) - Node module used to parse through the survey responses.
